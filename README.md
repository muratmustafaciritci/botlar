# Türkçe Sanal Asistan (Geliştirilmiş Sürüm)

Bu depo, Türkçe konuşan bir sanal asistanın çevrimdışı çalışan, pratik bir sürümünü içerir.

## Özellikler

- Selamlama ve doğal Türkçe yanıtlar
- Saat ve tarih söyleme
- Basit matematik hesaplama
- Not alma ve notları listeleme
- Motivasyon mesajları

## Çalıştırma

```bash
python assistant.py
```

## Test

```bash
python -m pytest -q
```
